RED:
- 命令: `.tools/uv-venv/bin/uv run --extra dev pytest tests/integration/oauth -q`
- 结果: 4 个测试失败。
- 失败摘要: `easyauth.applications.oauth` 尚不存在；`/oauth/token` 返回 404；OAuth access token 尚不能进入权限查询路径。

实现摘要:
- 新增 `OAuthClientBinding`，一个 DOT `Application` 只能绑定一个 EasyAuth `App`。
- 新增 `OAuthClientService.create_client`，创建 confidential client credentials OAuth client，并只返回一次性明文 client secret。
- 新增 `OAuthClientService.authenticate_access_token_for_api`，用 DOT access token checksum 解析有效 access token，并转换为与静态 token 相同的 `AppPrincipal`。
- `/api/v1` Bearer 认证先尝试静态 token，失败后尝试 OAuth access token；禁用 App 返回 EasyAuth 统一 403。
- 新增 `/oauth/token`，由 DOT `TokenView` 处理，保持 OAuth2 标准错误 JSON。
- 权限查询审计 metadata 可区分 `static_token` 与 `oauth_client`，未记录明文 client secret、静态 token 或 access token。
